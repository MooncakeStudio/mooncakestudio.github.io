Coffee Time

Coffee Time es un juego arcade multijugador realizado en Phaser3 y pensado para jugar en línea. ¿Quién logrará ser el empleado del mes en Buroun Kōhi? Este proyecto hace uso de una base de datos SQL conectada a través de un servidor hecho en Java y manipulada mediante peticiones API REST.
